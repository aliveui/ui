## Features

- A styled button component built on top of the shadcn Button primitive
- Custom gradient overlay background effect
- Precise shadow styling for a modern, polished look
- Maintains all functionality of the original Button component
- Sized appropriately for tight UI spaces
- Responsive touch targets
